# 🌍 Geo Learn

Geo Learn is a React-based web application designed to help users learn and understand different geometric shapes in an interactive and visual way.

---

## 🚀 Features

* 📐 Learn shapes like Square, Rectangle, Triangle, Parallelogram, Trapezium
* 🎯 Simple and user-friendly interface
* 📊 Interactive learning experience
* 🧠 Concept-based structure (React components, hooks, routing, forms)

---

## 🛠️ Tech Stack

* React.js
* JavaScript (ES6+)
* HTML5 & CSS3

---

## 📂 Project Structure

```
src/
 ├── pages/
 ├── shapes/
 ├── components/
 └── App.js
```

---

## ▶️ Getting Started

### Install dependencies

```
npm install
```

### Run the app

```
npm start
```

👉 Open in browser: http://localhost:3000

---

## 📦 Build for Production

```
npm run build
```
